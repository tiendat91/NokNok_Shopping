# NokNok WEBSITE
NokNok - Website mua bán thiết bị điện tử (ASP .NET CORE API/ C# .NET 6)

## Main Features

+ Jquery Ajax 
+ Store data in Session
+ Cookie Authorization Authentication (Client)
+ OData support using ASP.NET Core
+ Call and get API
